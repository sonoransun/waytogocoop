# Screenshots

Place PNG screenshots here. Referenced from the top-level `README.md` and
`CONTRIBUTING.md`.

Suggested captures (match the filenames below when replacing placeholders):

| File                      | How to capture |
|---------------------------|----------------|
| `viewer-2d.png`           | Python `/viewer`, heatmap view, any material pair, default twist. |
| `viewer-3d.png`           | Python `/viewer`, 3D Surface view mode. |
| `fourier.png`             | Python `/fourier`, showing the FFT heatmap and peaks table. |
| `proximity-3d.png`        | Python `/proximity3d`, isosurface rotated to show the `z=0` and ξ_prox annotations. |
| `rust-desktop-2d.png`     | Rust app 2D mode, axes and colorbar visible. |
| `rust-desktop-3d.png`     | Rust app 3D mode with wireframe toggled on. |
| `rust-desktop-menu.png`   | Rust app menu bar expanded (Help → About...). |

To capture from the Rust app press **Ctrl+S**; PNGs land in the current
working directory. From the Python app use Plotly's camera icon on any graph.
